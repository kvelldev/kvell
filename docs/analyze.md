# 分析

## 0. そもそもサービス企画の思考法とは?

個人開発のコアは、「自分だったらこのサービスこうするけどな」とか、「こういうサービスあったら使うけどな」だと思う。
B層というものの解像度をもっと上げる必要がある。「B層は書き込まない」の真偽は、現時点ではそれを判断するにはB層に対する解像度が低すぎる。では、B層とは何か？B層とは、ペルソナであり、そしてなにより自分自身だ。**「このサービスは、自分だったら書き込むか？」**をもっともっと考える必要がある。そして結論が「書き込む」なんだったらそれを信じて突っ走るしか無い。（なぜなら、どんなにマーケの勉強をしても、市場調査をしても、消費行動分析をしても……自分の感覚以外では「それを信じてアプリ開発をしてリリースまで突っ走る」というほどの確信を持てる状態にはならない。なぜなら無給だから。「調査の結果、なんか行けそうっぽいです（自分は使わないけど）」じゃあ「じゃあ自分が作る必要ないや」となってしまう。サラリーマンなら「まあ流行らなくても金もらえるし作るか」となるけど。）

逆に言えば、「自分だったら使わねえけどな」ってアプリを作ろうとしてるんだとしたら、それはサンクコストに目が眩んで自分を誤魔化そうとしてるってことだ。
但し、「今の企画なら使わないけど、こうすれば使うのでは？」という方向の思考は問題ない。問題なのは、「使わないけどな」と思いながらUI案をブラッシュアップしたり、レスポンスを改善したりして枝葉に迷い込んでしまうことだ。百の調査より一の「使いたい、楽しそう！」という自分の感覚。そして、その一を洗練するための百の分析と調査。ここの順番を間違っちゃいけない。
自分は、「荒れてない櫻坂板」が欲しかったんだと思う。それから、あわよくば井上初恋スレのような「祭り」を眺めたい。
で、「そもそも2chって……？」と調べていったら匿名掲示板そのものに惹かれていった。インターネットと技術でしか生まれない特殊な空間。それをみんなが使って、色々なコンテンツが生まれてくる。それを自分が作れたら……そんな気持ち。まとめサイトはずっと見てたし、そもそもそういう傾向はあったんだと思う。けど、少なくとも今の自分は「櫻坂板の住人」でしか無い。5chねらーと名乗るにはあまりにも他の板を知らなすぎる。そして他の板には興味がない。
だから、原点は常に「荒れてない櫻坂板」であるべきだろう。すべての要素はあくまでそれの実現のために実装される化粧であって、「チルい空間を作りたい」とか「焚き火のメタファーを実現するUXが欲しい」とかじゃない。そんな物はいらない。
良いものを作ろうとした結果、手段が目的になってきてる。真面目すぎるんだと思う。今自分はkvellの経営者なんだから、サラリーマン的な真面目さ(企画ありきで完成度高めようという思考)は必要ない。**「それって荒れてない櫻坂板のために必要なの？」**を常に問い続ける必要があると思う。

ひろゆきだって、最初は「落ちないあめぞう」を作りたかったんだろう。「日本中からアクセスが集まる、世界でも有数の匿名掲示板」を設計したわけじゃない。（結果としてそうなっていったのかもしれないけど。）
「安価のないBBS」でも、「昇格システムのあるBBS」でも、もっと言えば「B層のためのBBS」でも無いんだろう。とにかく「荒れてない櫻坂板」。結局B層ってのも、「荒れてない櫻坂板があればこういう人も書き込んでくれるよね」な気がする。だからB層→kvellじゃなくて、kvell（as a 荒れてない櫻坂板）→B層なんだろう。
「じゃあペルソナ分析とか『B層』みたいな考え方とか、必要無いんですか？」と言うと、そうじゃないと思う。そういうふうにゼロイチで考える必要はない。ただ、「荒れてない櫻坂板のために、それって必要なの？」だけは外しちゃいけないんだと思う。これと、それ以外のことと、それからリリース期限（賢者タイムになる前に作り切ること）。ここのバランス。

## 整理

整理したいのは、

- 「荒れてない櫻坂板」のために何が必要で何が必要でないのか?と、
- 井上初恋スレを生み出す土壌は?

この2つだ。
そのためには、まずは「荒れてない」の定義をする必要がある。

### 「荒れてない」とはなにか?

まずは「荒れてる」の定義を考えよう。

- 何を書き込んでも(ネガティブな意見でもポジティブな意見でも)批判コメントが付く
- 何を書き込んでも「他の板の奴らが工作してるから」みたいな陰謀論になってしまう
- 板のメインの話題に関して、ネガティブなコメントが多い。「〇〇ってブスじゃね?」など。
- 古参が新規を叩いている
- 他の板に比べてこの板は駄目だ...みたいな悲観論が多い
- 他の板と比べてこの板はマシ...みたいな比較論が多い
- 板と関係のない話題がたくさんある
- 特定のユーザーが連投している(「コテハン」が暴れている)
- 対立を煽るような扇動的なスレが多く立っている

→つまり、上記のような要素を何かしらの方法で(システム的、あるいはUI/UX的に etc.)排除できれば「荒れてない板」を実現できる。
